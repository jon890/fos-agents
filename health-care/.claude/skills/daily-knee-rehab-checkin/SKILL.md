# daily-knee-rehab-checkin

매일 아침 무릎 재활 체크인 메시지와 오늘의 보수적 재활 운동 세트를 한국어로 생성한다.

## 입력

다음 순서로 읽는다:

- `health-care/private/conditions/knee-patellar-instability/current-context.md` — 개인 현재 상태 (private)
- `health-care/config/knee-running-recovery-plan.md` — 공개 단계별 회복 플랜
- `health-care/config/knee-rehab-exercise-sets.md` — 공개 단계별 아침 운동 세트
- `health-care/config/public-health-care-policy.md` — 개인정보 경계

private 데이터가 없으면 공개 회복 플랜만 사용하고 컨텍스트 파일을 불러올 수 없음을 명시한다.

## 출력 형식

Discord 메시지로 간결하게 작성한다.
마크다운 표 없음.
Discord 2000자 제한 안에 맞춘다.

포함 항목:

1. 오늘의 체크인/리마인더임을 안내
2. 현재 기록 기준 단계를 조심스럽게 기술 — 예: "현재 기록 기준으로는 0단계/1단계에 가깝습니다"
3. 오늘의 재활 운동 세트 4-6개 — 각 운동은 횟수/세트/주의점 포함
4. 오늘 안전하게 할 수 있는 일상 행동 2-4가지
5. 오늘 피해야 할 행동
6. 중단 기준
7. 재진/상담 기준
8. 내일 계획에 도움이 될 짧은 질문 1개

## 운동 세트 선택 규칙

`current-context.md`의 최신 경과를 우선한다. 확실하지 않으면 낮은 단계 세트를 선택한다.

- 통증·붓기·빠질 듯한 느낌이 남아 있거나 최근 악화가 있으면 `세트 A`.
- 평지 보행은 가능하지만 계단 하강이 조심스럽거나 불안정하면 `세트 B`.
- 일반 계단 하강이 0~2/10이고 붓기/열감이 없으나 빠른 계단 하강은 아직 테스트하지 않았으면 `세트 C`.
- 평지 40~60분 보행과 계단 하강이 안정적이고 달리기 복귀 전 점검 단계면 `세트 D`.

2026-05-26 현재 기록처럼 "일반 계단은 무리 없이 내려오지만 여러 계단을 빠르게 내려가기는 아직 미테스트"인 경우에는 `세트 C`를 기본으로 하되, 빠른 계단 하강은 피해야 할 행동에 포함한다.

아침 메시지에 운동을 너무 많이 넣지 않는다. `세트 C`라도 피로도나 전날 활동량이 높으면 `세트 B` 수준으로 낮춘다.

## 안전 규칙

진단·처방·의학적 확정 표현을 하지 않는다.
한국어 안내에서도 의사 지시처럼 들리는 표현(예: "~하세요") 대신 제안형(예: "~하는 것이 좋습니다", "~을 권장합니다")을 유지한다.

문서화된 기준 충족 + 의료진/물리치료사 확인 권장 없이는 러닝 복귀를 안내하지 않는다.

운동 안내는 재활 참고 정보이며 처방처럼 단정하지 않는다. "오늘은 이 정도를 권장합니다", "불편하면 여기서 멈추는 게 좋습니다"처럼 보수적으로 쓴다.

중단 기준은 항상 포함한다:

- 빠질 듯한 느낌(giving-way sensation)
- 찌르는 통증
- 붓기/열감 증가
- 덜그럭거림 증가
- 잠김/걸림(locking/catching)
- 다음날 악화

증상이 재평가 기준보다 악화된 것으로 보이면 기다리지 말고 병원 재평가를 권장한다.

"참고 견뎌라" 조언보다 보수적 축소 조언을 우선한다.

개인 식별자(Discord/서버/사용자 ID, 정확한 병원 식별자, raw 의무 기록 세부 내용)를 노출하지 않는다.
