# PRD — cooking

cooking 워크스페이스의 제품 범위와 MVP 기능 명세.

## 1. 목적

집에서 만들 요리의 조사 과정을 반복 가능하게 만든다.
핵심은 레시피 수집이 아니라, 첫 시도에서 실패 가능성을 낮추는 재료 선택과 조리 순서 압축이다.

## 2. 현재 MVP 타깃

- 잠봉뵈르 샌드위치
- 시판 냉면 육수 + 물냉면 + 양념 돼지갈비 조합

## 3. 기능 목록

- 재료 알아오기: 핵심 재료, 대체재, 구매처 후보, 보관 조건
- 재료 비교하기: 맛, 접근성, 가격, 실패율 기준 비교
- 레시피 압축하기: 1회차용 간단 순서와 실패 포인트
- 영상 수집하기: 한국어 실전 영상, 원조/해외 참고 영상, 짧은 확인용 영상
- 장보기 리스트 만들기: 인원별 수량과 남는 재료 활용법

## 4. 산출물

- `data/<menu-slug>/ingredients.md`
- `data/<menu-slug>/comparison.md`
- `data/<menu-slug>/recipe.md`
- `data/<menu-slug>/videos.md`
- `data/<menu-slug>/shopping-list.md`

## 5. 의도적으로 안 하는 것

- 영양·의학 조언
- 식품 안전을 추측으로 단정
- 현재 가격을 확인하지 않고 최신 가격처럼 표현
- 개인 구매처나 주소가 들어간 내용을 공개 자료로 발행

## 6. 성공 기준

- 사용자가 장보기 전에 바로 볼 수 있는 짧은 추천 조합이 있다.
- 첫 조리 때 필요한 수량과 순서가 명확하다.
- 링크와 재료 후보가 나중에 갱신 가능하게 분리되어 있다.
