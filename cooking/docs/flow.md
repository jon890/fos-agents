# Flow — cooking

cooking 워크스페이스의 기본 작업 흐름.

## 1. cooking-research

입력은 메뉴명과 선택 조건이다.
예시는 `잠봉뵈르`, `2인분`, `첫 시도`, `국내 구매 쉬운 재료` 같은 형태다.

흐름:

1. 메뉴의 정체성과 핵심 재료를 확인한다.
2. 국내에서 구하기 쉬운 재료 후보를 모은다.
3. 가격, 접근성, 맛, 실패율 기준으로 비교한다.
4. 첫 시도용 추천 조합을 정한다.
5. 조리 순서를 5단계 안팎으로 압축한다.
6. 영상과 레시피 링크를 분리해 저장한다.
7. 장보기 리스트를 인원수 기준으로 정리한다.
