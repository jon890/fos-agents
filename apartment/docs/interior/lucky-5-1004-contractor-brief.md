# 구리 럭키아파트 5동 1004호 인테리어 업체 상담 브리프

- 기준일: 2026-05-13
- 목적: 업체 연락/실측 요청 시 동일한 범위로 견적을 받기 위한 요약

## 기본 정보

- 구축 20평대 아파트
- 스타일: 화이트/베이지 + 과하지 않은 우드 포인트
- 욕실: 1곳. 세컨 욕실/부부욕실 없음
- 예산 기준: 기본 3천만원 전후에서 시작하되, 1차 견적상 부가세 별도 확인으로 실제 총예산 상한은 3,500만원 전후까지 열어둔다. 발코니 마감/경계부 문/샷시 보수/제작가구 등 옵션은 분리 견적 요청
- 시스템에어컨: 설치하지 않음

## 희망 공사 범위

- 도배
- 바닥: 실제 600각 포세린 타일보다는 포세린/스톤 타일 느낌 나는 강마루/마루 계열 희망. 올철거 후 거실·주방·방·복도 전체를 베이지/크림/연그레이지 톤으로 통일감 있게 검토. 실시공 면적 17~20평 전후 가정, 중급 제품 + 철거/폐기/바탕 정리 포함 300~450만원 선을 1차 현실 범위로 봄
- 욕실 전체 공사 + 라인유가 + 젠다이 + 거울장
- 싱크대
- 조명
- 문/몰딩
- 공용부 천장 덴조/천장 목공은 필요 가능성이 높으므로 별도 금액과 포함 범위 확정 필요. 1차 구두 기준 공용부 천장만 약 100만원 전후로 메모
- 전기/콘센트/스위치
- 중문 설치
- 거실 베란다는 비확장 유지
- 거실 발코니 마감 개선 + 거실-발코니 경계부 문/틀 상태 확인
- 거실 외창 샷시는 유지/부분 보수 중심 검토
- 안방 일자형 붙박이장
- 작은방 스타일러 자리 확보
- 작은방 작은 컴퓨터 책상 배치
- 작은방은 기존 IKEA 옷장 활용을 기본안으로 보고, 제작 붙박이장은 옵션 견적으로만 비교
- 안방 앞/화장실 입구 작은 화장대 + 거울/조명 + 드라이기용 콘센트

## 제외/유지 방향

- 시스템에어컨 제외
- 전체 샷시 교체 제외, 기존 샷시 유지/부분 보수 중심으로 검토
- 거실 확장은 기본안에서 제외하고 예외 비교안으로만 확인
- 안방 쪽 베란다 비확장
- 보일러는 약 6년 전 교체가 맞으면 유지, 분배기/배관/누수/난방 순환만 점검

## 업체에 꼭 확인할 질문

1. 구리 럭키아파트 동일 평형에서 거실 비확장 유지 시 많이 하는 발코니 마감 방식
2. 기존 거실-발코니 경계부 문/틀을 유지할 수 있는지, 교체한다면 폴딩도어/터닝도어/슬라이딩 도어 중 무엇이 적합한지
3. 기존 거실 외창 샷시를 유지해도 단열/결로/소음 문제가 없는지
4. 전체 샷시 미교체 시 필요한 부분 보수 범위
5. 타일 느낌 강마루/마루 제품 샘플과 바닥 수평/거실-발코니 경계부 단차/걸레받이·문턱 간섭. 자재비/시공비/철거·폐기비/바탕 보정/걸레받이/문턱 마감 포함 여부 분리 견적
6. 중문은 3연동 슬라이딩/원슬라이딩/여닫이 중 어떤 타입이 평면에 적합한지
7. 안방 앞/화장실 입구 화장대 설치 시 통행폭/문 간섭/습기/콘센트 가능 여부
8. 욕실에 젠다이와 거울장을 함께 설치할 수 있는지, 위치/폭/깊이/문 열림 간섭이 없는지
9. 욕실 라인유가 시공 시 배수 위치, 바닥 구배, 방수 마감, 트랩/커버 청소 구조에 문제가 없는지
10. 세탁실에 LG FX24ENE 세탁기 + RD21ESE 건조기 직렬 설치가 가능한지
11. 보일러/분배기/배관 상태 점검 가능 여부
12. 기존 에어컨 배관/실외기 위치 사용 가능 여부
13. 냉장고장 제작 시 LG J825MEE042 냉장고 치수와 급수 연결, 방열 여유, 문 열림 각도를 반영할 수 있는지
14. 싱크대 하부장에 12인용 빌트인 식기세척기 기준으로 설치 공간과 급수·배수·전원·걸레받이·도어 간섭을 반영할 수 있는지
15. 아일랜드 식탁/장에 밥솥·에어프라이어·전자레인지 배치와 로봇청소기 도킹 공간을 함께 반영할 수 있는지
16. 작은방에 작은 컴퓨터 책상 + 스타일러 + 기존 IKEA 옷장을 함께 배치할 경우 콘센트, 랜선, 문열림, 의자 후퇴 공간 간섭이 없는지
17. 작은방 제작 붙박이장을 옵션으로 넣을 경우 기존 IKEA 옷장 활용안 대비 절감/효용 차이가 충분한지
18. 공용부 천장 덴조/천장 목공은 실제로 필요한지, 약 100만원으로 보면 되는지, 거실/주방/복도 중 어디까지인지, 방 천장은 제외인지, 목상/석고/퍼티/다운라이트 타공/커튼박스/도배 연계가 어디까지 포함인지
