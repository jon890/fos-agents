# 구리 럭키아파트 5동 1004호 인테리어 레퍼런스 누적 노트

## 목적

아침 오늘의 인테리어 추천에서 발견한 후보 중, 나중에 업체 미팅/견적 요청/스타일 결정에 다시 볼 만한 링크를 누적한다.

- 결정 확정 문서는 `lucky-5-1004-interior-decisions.md`
- 이 문서는 레퍼런스/업체/아이디어 후보 보관용
- 결정으로 승격하려면 사용자 확인 후 의사결정 노트에 `D-00X`로 반영

## 현재 핵심 기준

- 대상: 구리 럭키아파트 5동 1004호
- 예산: 기본 3,000만원 전후, 필요 시 증액 가능
- 스타일: 화이트/베이지 + 우드 포인트
- 현재 선택 방향:
  - 거실 베란다 비확장 유지
  - 전체 샷시 교체는 우선 제외, 기존 샷시 유지/부분 보수 중심
  - 벽과 천장은 같은 벽지로 통일, LX 베스띠 실크 페인트 계열 후보 우선
  - 문/문선은 화이트 계열 + 9mm 문선
  - 바닥은 실제 포세린 타일이 아니라 베이지/크림/연그레이지 톤의 타일 느낌 강마루/마루 계열
  - 우드 포인트는 중문/아일랜드/가구 중심으로 과하지 않게 사용
- 우선 관심사:
  - 거실 베란다 비확장 유지
  - 발코니 마감 개선과 거실-발코니 경계부 문/틀
  - 기존 샷시 유지/부분 보수 가능 여부
  - 욕실/주방/조명/문/몰딩/전기
  - 안방 붙박이장
  - 작은방 작업실/모션데스크 배치
  - 전체 타일 느낌 강마루 후보와 실제 샘플/견적 비교

## 현재 선택 기준별 레퍼런스 정리

기준일: 2026-06-06

이 인덱스는 누적 레퍼런스를 현재 선택 방향에 맞춰 빠르게 거르기 위한 용도다. 기존 본문은 히스토리와 원문 링크 보존을 위해 삭제하지 않는다.
하나의 레퍼런스가 여러 분류에 중복될 수 있다. 예를 들어 톤은 좋지만 바닥재가 다르거나, 비확장 판단에는 좋지만 폴딩도어는 기본 선택이 아닌 경우가 있다.

현재 선택과 직접 맞거나, 업체 상담 때 바로 꺼내기 좋은 레퍼런스다.

### B. 현재 선택과 일부 어긋나므로 보조로만 볼 레퍼런스

확장, 전체 샷시, 고급 브랜드 패키지, 올확장, 과한 제작가구가 중심인 사례다. 지금 방향의 메인 레퍼런스로 쓰지 않고, "이렇게 하면 예산과 리스크가 커진다"는 비교용으로만 본다.

- 거실확장/전체 샷시 중심: R-005, R-011, R-012, R-014, R-015, R-018, R-019, R-020, R-022, R-026, R-027, R-029, R-031, R-032, R-033, R-036, R-042, R-051, R-052, R-065, R-066, R-078, R-081
- 3,000만원 기본안보다 상향 사양에 가까움: R-005, R-011, R-012, R-020, R-029, R-040, R-042
- 현재 바닥 선택과 다름: 실제 포세린 전체 시공, 장판 중심, 오크 마루 복귀 트렌드는 직접 채택하지 않고 비교용으로만 본다. 관련: R-013, R-022, R-027, R-075, R-078, R-081, R-089, R-090, R-091
- 우드 비중이 커질 수 있음: R-076, R-092, R-093, R-096은 밝은 오크 원칙은 참고하되, 하부장/가구 우드 면적을 20~30% 안에서 조절한다.
- 폴딩도어 중심 사례: R-010, R-034, R-050, R-070, R-075, R-087, R-091은 개방감 참고용. 기본 선택은 아니며 단열·기밀·레일 청소·하자 책임을 따로 확인한다.

### C. 검증 후 보조로 볼 레퍼런스

본문 추출 실패, 광고성 강함, 가이드형 콘텐츠, 단지 직접성이 낮은 자료다. 결정을 바꾸는 근거가 아니라 질문 리스트 보조로만 사용한다.

- 본문/사진 확인 보류: R-007
- 일반 가이드형 또는 광고성 보조자료: R-024, R-030, R-038, R-047, R-054, R-058, R-059, R-064, R-068, R-069, R-071, R-072, R-079, R-080, R-082, R-083, R-084, R-088, R-090, R-094, R-095, R-096, R-097, R-098, R-099
- 동일 단지/평형 직접성이 낮아 부분 참고만: R-004, R-006, R-011, R-012, R-014, R-015, R-018, R-020, R-027, R-028, R-029, R-031, R-032, R-033, R-036, R-040, R-045, R-046

### D. 당장 상담/샘플 때 확인할 비교축

- 바닥: 사하라 라이트/이모션 블랑/KCC 스톤 스퀘어/마뷸러스 젠을 같은 조명 아래 샘플로 보고, 총액 300~450만원 기본선과 450~600만원 상한선을 분리한다.
- 벽지: 프레스코·스톤·강한 회벽 질감은 제외하고, 베스띠 실크 페인트 계열의 유백색/화이트/내추럴 페인팅 후보를 벽+천장 통일 기준으로 본다.
- 샷시: 전체 교체 레퍼런스는 예외 옵션이다. 현장에서는 거실 외창 상태, 외풍, 결로, 레일, 실리콘, 잠금장치 보수 가능성을 먼저 확인한다.
- 발코니: 확장 레퍼런스보다 비확장 마감, 경계부 문/틀, 바닥 타일, 벽면 페인트, 수납장, 결로 관리 루틴을 우선 본다.
- 주방 우드: 1안/3안처럼 중문+아일랜드 중심 우드가 기본. 하부장 전체 우드는 샘플 비교 후 우드 비중이 과하면 낮춘다.
- 욕실: 1곳 기준 중급 500만원대 안팎을 현실선으로 두고, 600각/300x600, 젠다이, 거울장, 라인유가, 환풍기, 선반 물고임을 확인한다.

  - 욕실/세탁실은 4000K 단일로 단순화 가능.

### R-084. 월간 THE LIVING - 화이트 & 아이보리 인테리어 색 가이드 (순백/웜화이트/아이보리 구분)

- 링크: http://www.theliving.co.kr/news/articleView.html?idxno=22415
- 출처: 인테리어 매거진 (월간 THE LIVING)
- 적합도: 높음 (신규 Q-039 직접 연결, config toneReferenceFocus 검증)
- 이유:
  - 순백 화이트("밝고 화사, 수평·수직 확장 효과") / 웜화이트("차분하고 따뜻") / 아이보리("편안하고 부드러움")의 분위기 차이를 명시.
  - "밝은 오크 바닥재 + 화이트 몰딩 = 모던", "원목 가구·우드 마감 + 웜화이트 = 포근"으로 밝은 오크 조합을 권장 → config woodTone(밝은 오크/애쉬)과 일치.
  - 차가운 화이트를 따뜻하게 만드는 방법(우드 가구·패브릭·원목 소품)을 정리해 초보자 벽 톤 결정의 기준이 됨.
- 참고 포인트:
  - 벽/천장은 화이트 70~80% 전제, 밝은 오크 바닥/가구 20~30% 포인트 조합 권장.
  - 너무 밝은 순백이 부담되면 아이보리/웜화이트로, 단 우드까지 누런 톤이면 전체가 누렇게 떠 보일 위험.
  - 무몰딩/마이너스 몰딩 + 밝은 오크 바닥 + 화이트 몰딩 조합.
- 주의:
  - 매거진 일반론 가이드라 구리 럭키아파트 직접 사례 아님.
  - 같은 벽지도 조명 색온도(Q-030)에 따라 누렇게/푸르게 보이므로 색온도와 같이 봐야 함.

### R-085. 블라인드 - 얼음정수 냉장고 급수 배관 설치 후기 (바닥 노출 실패 + 하부/다용도실 숨김 대안)

- 출처: 블라인드 직장인 커뮤니티 / 실사용 후기 + 댓글
- 적합도: 높음 (Q-021 직접 연결, D-024 LG J825MEE042 급수 후속)
- 이유:
  - 아일랜드 주방에서 LG 오브제 냉장고 급수 배관을 "주방 바닥으로 노출" 설치 후 발매트로 덮는 C 실패 사례가 구체적.
  - 대안: 다용도실 방향 연장 또는 싱크대 하부 통과로 숨김 가능.
  - "커플러 사용 시 세월이 흐르면 누수 가능" → 연결부 최소화가 노출 여부보다 더 중요.

- 참고 포인트:
  - 냉장고장 위치(D-024)를 싱크대 급수와 가깝게 잡으면 노출/타공 구간 자체가 짧아짐.
  - 견적서에 "급수 분기 위치 + 배관 경로 + 연결부 개수 + 누수 테스트"를 항목으로 요청.
  - R-043(온사랑집수리 급수 노출 최소화)과 묶어 숨김 시공 + 점검 접근성 균형 검토.
- 주의:
  - 익명 커뮤니티 글이라 개별 시공 진위는 별도 검증 필요.
  - 신축/아일랜드 주방 기준 의견이 섞여 있어 구리 럭키 구축 + 비아일랜드 조건과 다를 수 있음.

### R-087. LX Z:IN - 신혼부부를 위한 화이트 24평 (안양 한마음아파트, 민자 천장 + 밝은 오크 포인트)

- 링크: https://www.lxzin.com/styling/expert-style/detail/218
- 출처: LX Z:IN 브랜드 시공사례
- 적합도: 높음 (오늘 화이트/우드 톤 집중 + Q-040 천장 몰딩 직접 연결)
- 화이트/우드 톤 평가:
  - 비율: 화이트 약 80% + 우드 약 20% (벽·천장·신발장·싱크대 화이트 통일, 드레스룸/침대만 우드 포인트)
  - 우드 색감: 밝은 오크 (기존 어두운 체리색 장을 밝은 오크 + 미러 도어로 교체)
  - 과하지 않은지: 과하지 않음. 우드는 드레스룸·침대 포인트로만, 카페풍 아님.
- 이유:
  - 20평대(약 24평) 화이트 톤 통일로 개방감을 만든 사례. 예산 2,000~3,000만원으로 우리 기본안과 겹침.
  - "민자 천장"(평평한 천장 마감) 적용 → 오늘 신규 Q-040(천장 몰딩 방식) 비교 근거.
  - 거실/주방 폴딩도어, 카운터식 싱크대, 최소 수납으로 좁은 평형을 넓게 쓰는 전략.
- 참고 포인트:
  - 화이트 마감 통일 + 밝은 오크 포인트의 균형(70~80% : 20~30%)을 그대로 따라가기 좋은 톤.
  - 어두운 체리색 가구 → 밝은 오크 교체가 톤을 얼마나 밝히는지 비교 사례.
  - 현관 패턴 타일 + 화이트 신발장 조합.
- 주의:
  - 브랜드 시공사례라 상세 자재명·AS·방수/단열 공정 설명은 제한적.
  - 폴딩도어는 우리 집 거실 비확장 유지 방향과는 별개 대안(단열/기밀 별도 확인).

### R-088. LX Z:IN - 몰딩 선택부터 시공·교체까지 가이드 (일반/마이너스/무몰딩 비교)

- 링크: https://www.lxzin.com/styling/style-guide/detail/6979
- 출처: LX Z:IN 스타일 가이드
- 적합도: 중상 (오늘 신규 Q-040 천장 몰딩 방식 직접 비교 근거)
- 이유:
  - 일반(평)몰딩 / 마이너스몰딩 / 무몰딩의 정의·장단점·비용 차이를 한 번에 비교.
  - 마이너스·무몰딩은 일반 대비 "두 배 이상" 비용, 무몰딩은 "평탄한 벽체 마감 작업이 필수" → 구축 콘크리트/조적벽에서 목공 추가 비용 발생 근거.
  - 마이너스몰딩은 천장-벽 틈에 간접조명/포토레일 매입 가능, 천장 정리로 넓어 보이는 효과.
- 참고 포인트:
  - 평몰딩(저렴·시공 쉬움) vs 마이너스몰딩(간접조명·넓어 보임·2배+) vs 무몰딩(가장 깔끔·시공난이도 최상·하자 리스크)를 화이트 톤 기준으로 비교.
  - 전체 교체가 부담되면 도장/필름으로 기존 몰딩 톤만 화이트로 정리하는 대안도 제시.
  - 무몰딩/마이너스몰딩은 목공 실력 의존도가 커서 전문가 시공 권장.
- 주의:
  - 브랜드 가이드라 구체 평당 단가는 없음. 견적 시 업체별 목공비 분리 확인 필요.
  - 9mm 문선(Q-019 결정)·걸레받이 색상 통일과 함께 톤을 맞춰야 따로 놀지 않음.

### R-089. 집꾸미기 - 21평 의정부 화이트&베이지&우드 가성비 스타일링 (벽지·장판만, 북유럽풍)

- 링크: https://mystylezip.com/apartment/article/47021/
- 출처: 집꾸미기 스타일
- 적합도: 중상 (화이트 70~80% 기반 + 우드 포인트 톤 레퍼런스)
- 화이트/우드 톤 평가:
  - 비율: 화이트/베이지 약 70~80% + 우드 포인트
  - 우드 색감: 밝은 톤 (소파·TV장·주방 조리대·침실 협탁에 밝은 우드)
  - 과하지 않은지: 과하지 않음. 미니멀+따뜻한 북유럽식, 카페풍 아님.
- 이유:
  - 21평 비확장 + 벽지·장판만 최소 시공으로 화이트&우드 톤을 만든 가성비 사례.
  - 시공보다 가구/소품 톤 조합 비중이 커서, 시공 후 가구 단계 톤 매칭 참고에 유용.
- 참고 포인트:
  - 큰 시공 없이도 화이트 벽 + 밝은 우드 가구로 톤을 만드는 방법.
  - 주방 우드 조리대 포인트, 욕실 화이트 기조 + 소품 포인트.
- 주의:
  - 시공 사례가 아니라 스타일링 중심. 평형(21평)·지역(의정부)이 우리와 다름.
  - 비용·자재명·AS 정보 없음. 톤 감각 참고용.

### R-092. 하이마트 하이홈스 - 첫 신혼집 24평 화이트&우드 (화이트 60 : 우드 40)

- 출처: 하이마트 하이홈스 매거진 (스타일링 후기)
- 적합도: 중상
- 화이트/우드 톤 평가:
