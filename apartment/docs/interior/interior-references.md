## 목적

아침 오늘의 인테리어 추천에서 발견한 후보 중, 나중에 업체 미팅/견적 요청/스타일 결정에 다시 볼 만한 링크를 누적한다.

- 결정 확정 문서는 `lucky-5-1004-interior-decisions.md`
- 이 문서는 레퍼런스/업체/아이디어 후보 보관용
- 결정으로 승격하려면 사용자 확인 후 의사결정 노트에 `D-00X`로 반영

## 현재 핵심 기준

- 대상: 구리 럭키아파트 5동 1004호
- 예산: 기본 3,000만원 전후, 필요 시 증액 가능
- 스타일: 화이트/베이지 + 우드 포인트
- 우선 관심사:
  - 거실 베란다 확장
  - 단열 강화
  - 거실 샷시 교체 여부
  - 욕실/주방/조명/문/몰딩/전기
  - 안방 붙박이장
  - 작은방 작업실/모션데스크 배치
  - 공용부 600각 타일 + 방 마루 조합 가능성

- 이유:
  - 24평 직접 시공 사례에서 거실 등박스 중앙 4인치 주백색 + 모서리 매립등 + 아트월 3인치 전구색을 별도 회로로 독립 제어한 구성이 명시됨.
  - 침실은 슬림 사각 엣지등, 주방은 메인등 없이 매립등 균등 배치, 욕실은 방습형 매립등으로 공간별 사양 분리.
  - 내부 스위치/콘센트 전 영역 화이트 커버 교체를 시공 과정에 포함해 견적서 표준 형태로 응용 가능.
- 참고 포인트:
  - 등박스 중앙(주백색 4인치) vs 모서리(전구색) 회로 분리 — 거실 1구가 아닌 다구 스위치 견적 요청
  - 침실 슬림 엣지등 + 주방 매립등 균등 배치 + 욕실 방습형 매립등 영역별 사양 분리
  - 스위치/콘센트 전 영역 화이트 매립 커버 교체 견적 항목 표준화
- 주의:
  - 업체 포트폴리오라 비용·AS·하자 대응 정보는 부족.
  - 직접 단지가 아니므로 럭키아파트 5동 천장 콘크리트 두께/등박스 형태에 따라 적용 가능성 별도 확인 필요.

### R-058. 꿈이고 - 구축 아파트 리모델링 평당 단가 비교 (20평대 A·B·C 3안 + 우선순위)

- 링크: https://kkumigo.com/interiortip/%EA%B5%AC%EC%B6%95-%EC%95%84%ED%8C%8C%ED%8A%B8-%EB%A6%AC%EB%AA%A8%EB%8D%B8%EB%A7%81/
- 출처: 인테리어 정보 블로그 (꿈이고)
- 적합도: 높음 (Q-016 직접 연결)
- 이유:
  - 20평대 구축 평당 단가를 A 100만(총 2,000만)·B 150만(총 3,000만)·C 180~200만(총 3,600~4,000만) 3안으로 직접 제시.
  - 욕실 1곳 기본형 → 욕실 2곳 중 1곳 고급형 → 욕실 2곳 고급형+수입 제품으로 단계화한 견적 비교 기준선.
  - "욕실 → 바닥 → 주방 → 방문 → 중문 → 아트월" 우선순위 가이드 제공.

  - 욕실 2칸 견적 700~1,000만원 범위 검증 기준
  - 우선순위 가이드가 D-002 기본 포함 후보와 일치 → 예산 부족 시 중문/아트월 후순위 결정 근거
- 주의:
  - 정보 블로그 일반화 자료라 럭키아파트 직접 사례 아님 (±10~20% 편차).
  - 욕실 덧방은 단기 비용 절감되나 5~7년 내 누수·곰팡이 재발 위험. UBR이거나 2015년 수리분 노후 상태면 전체 철거+3차 방수 표준.

- 적합도: 높음 (Q-015 직접 연결, D-024 인덕션 후속과 직접 맞물림)

  - 일반 가정 콘센트 허용치 3,300~3,400W를 명시하고, 인덕션 출력별 공사 분기점을 정량적으로 제시.
  - 직결+전용 차단기(20A, 약 4,400W)와 단독 배선 공사(35A, 약 7,700W) 두 옵션을 정리.
  - 부실 시공(기존 라인에 차단기만 새로 달고 "전용선"이라 청구) 주의 포인트 명시.
- 참고 포인트:
  - 견적서 "인덕션 전용회로 공사" 항목 분리 + 차단기 용량 + 배선 굵기 + 분전반 여유분 확인 요청

  - 시공 중 분전반 사진·배선 라우트 사진 받기 조건을 계약서에 명시
  - LG J825MEE042 냉장고도 같은 주방 분전반 공유 → 인덕션 전용 + 식세기 전용 + 냉장고/아일랜드 가전 일반 회로 3분기
- 주의:
  - 사양 백과 자료이며 실측 사례 아님. 분전반 여유분·아파트 인입 전력 한도는 럭키 5동 현장 확인 필요.
  - 전기공사는 도배·타일 이전 단계라 누락 시 후공정 다시 뜯어야 하는 비용 가장 큼.

### R-060. 바른도어 - 중문 5타입 비교표 (초슬림·비대칭 양개스윙·3연동·원슬라이딩·ㄱ자 파티션)
