# 구리 럭키아파트 5동 1004호 인테리어 의사결정 요약

- 기준일: 2026-05-13
- 원본 의사결정 노트: `lucky-5-1004-interior-decisions.md`
- 목적: 지금까지 정해진 방향을 견적 상담/검토용으로 한눈에 보기

## 1. 기본 방향

- 대상: 구리 럭키아파트 5동 1004호
- 잔금일: 2026-09-09
- 입주: 인테리어 완료 후 입주 예정
- 욕실 수: 1곳. 세컨 욕실/부부욕실 없음
- 1차 예산: 약 3,000만원 전후에서 시작
- 현실 예산 상한: 엘스인테리어 1차 견적 기준 부가세 별도 확인. 작은방 붙박이 제외 + 공용부 천장 목공 추가 후 부가세 포함 약 3,178.7만원, 선호 마감 업그레이드와 예비비 포함 시 총예산 상한은 3,500만원 전후로 본다
- 스타일: 밝고 따뜻한 화이트/베이지 톤 + 과하지 않은 우드 포인트
- 전략: 기본 공사와 옵션 공사를 분리해서 견적 비교
- 예산 관점: 잘 수리해서 실거주하다가 향후 갈아타기까지 고려

## 2. 현재 결정된 큰 범위

### 기본 포함 후보

- 도배: 벽과 천장은 같은 벽지로 통일. 제품은 샘플 확인 전이며, LX 베스띠 실크 페인트 계열 3개 후보를 우선 검토
- 바닥재
- 욕실
- 싱크대
- 조명
- 문/몰딩: 9mm 문선 + 방문/도어 화이트 계열로 진행. 세부 화이트 톤은 벽지·걸레받이 샘플과 맞춰 확인
- 전기, 콘센트, 스위치

### 옵션/상태 확인 후 판단

- 거실 외창 샷시 교체
- 보일러 유지/점검
- 중문: 3연동 슬라이딩 + 모루/플루트 계열 유리로 결정
- 붙박이장/제작가구
- 시스템에어컨: 설치하지 않음으로 결정
- 안방 앞/화장실 입구 화장대

## 3. 거실 비확장

- 상태: 방향 변경
- 방향: 거실 베란다는 확장하지 않고 유지하는 쪽
- 이유:
  - 구리 럭키아파트는 거실 확장을 많이 하지 않는 단지로 보임
  - 확장 시 단열, 결로, 샷시, 난방 배관, 허가 리스크와 비용이 커짐
  - 3,000만원 기본안에서는 확장보다 욕실/주방/바닥/조명/중문/제작가구 완성도가 우선
- 대신 확인할 것:
  - 기존 거실-발코니 경계부 문/틀 상태
  - 발코니 바닥/벽/천장 마감 개선 범위
  - 폴딩도어/터닝도어/슬라이딩 도어 대안의 단열·기밀·소음 성능
  - 기존 거실 외창 유지 가능 상태
  - 동일 평형/동의 확장 선례 여부는 예외 옵션 판단용으로만 확인

## 4. 샷시

- 상태: 방향 결정
- 방향: 전체 샷시 교체는 우선 제외, 비확장 유지 전제에서 기존 샷시 유지/부분 보수 중심으로 검토
- 이유:
  - 전체 샷시 교체는 예산 부담이 큼
  - 거실 비확장 유지 시 거실 외창 교체 필요성이 낮아짐
  - 방/주방 쪽은 상태가 나쁘지 않다면 유지 또는 부분 보수로 충분할 수 있음
- 견적 요청 방향:
  - 전체 샷시는 유지 전제
  - 필요 시 실리콘, 레일, 잠금장치, 필름 등 부분 보수만 확인
  - 거실 외창 교체는 현장 상태가 매우 나쁘거나 확장안을 다시 선택할 때만 예외 옵션으로 확인
- 현장 확인:
  - 거실 외창 단창/이중창 여부
  - 페어유리/로이유리 여부
  - 창틀 뒤틀림, 외풍, 물샘, 결로 흔적
  - 개폐/잠금 상태

## 5. 보일러

- 상태: 유지 쪽으로 기울어짐
- 현재 정보: 약 6년 전 교체했다고 들음
- 판단:
  - 일반적인 보일러 교체주기는 10년 전후
  - 상태가 좋으면 10~15년까지도 사용 가능
  - 6년차가 맞다면 이번 인테리어 때 선제 교체할 필요는 낮음
- 방향:
  - 정상 작동이면 유지
  - 공사 전 설비기사 또는 업체를 통해 점검
- 확인할 것:
  - 제조년월/설치년월 사진
  - 모델명/용량
  - 온수 온도 안정성
  - 난방 순환 상태
  - 소음, 에러코드, 누수 여부
  - 분배기/배관 누수·녹·부식 흔적
  - 연통/배기통 체결 상태

## 6. 욕실

- 상태: 임시 결정
- 방향: 이번에 새로 공사하는 쪽에 무게
- 전제: 욕실은 1곳 기준. 세컨 욕실/부부욕실은 없으므로 2곳 합산 견적이나 메인/세컨 분리 견적은 제외
- 이유:
  - 2015년쯤 수리했더라도 이미 약 10년 전
  - 욕실은 입주 후 다시 공사하기 번거로움
  - 공실일 때 방수/배관/타일을 같이 보는 것이 유리
- 선호:
  - 베이지 계열
  - 600각 또는 300×600 타일 검토
  - 배수구는 라인유가
  - 환풍기 좋은 제품 고려
  - 욕실 선반/수납 필요
  - 젠다이 + 거울장 둘 다 설치하는 방향
- 선반/수납 방향:
  - 기본: 젠다이 + 거울장
  - 추가 옵션: 매립 선반 또는 코너 선반은 구조/비용 보고 선택
- 견적서 확인:
  - 철거 범위
  - 방수 횟수와 범위
  - 배관/설비 교체 범위
  - 라인유가 시공 가능 여부와 바닥 구배/트랩 청소 구조
  - 환풍기 제품명 또는 성능
  - 젠다이 위치/폭/깊이와 거울장 크기/문 열림
  - 선반/수납 위치와 방식, 물고임/청소/모서리 충돌 리스크

## 7. 바닥재

- 상태: 결정, 샘플 실물 확인 필요
- 현재 판단:
  - 거실과 방은 실제 600각 포세린 타일보다는 타일 느낌 나는 강마루/마루 계열로 진행
  - 목표는 우드 느낌을 줄인 베이지/크림/연그레이지 톤
  - 실제 타일의 차가움, 딱딱함, 줄눈 관리, 시공비 부담은 줄이는 방향
- 추천 방향:
  - 기본안은 스톤/타일 느낌 강마루
  - 고급 장판은 예산 절감 비교안으로만 확인
  - 실제 포세린 타일은 현관·욕실·주방 등 일부 포인트 구역에서만 검토
  - 전체 바닥을 같은 계열로 이어 통일감 확보
  - 2026-05-27 대화 기준, 올철거 후 거실·주방·방·복도 전체를 포세린 타일 느낌 강마루로 시공하는 안을 1순위 견적으로 본다
  - 실시공 면적은 24평 전체가 아니라 욕실/발코니/현관 등을 제외한 약 17~20평 전후로 가정
  - 중급 제품 기준 총액 300~450만원 선이면 현실적인 선택지로 보고, 상급 제품/바탕 보정 많을 경우 450~600만원대까지 열어둔다
- 샘플 후보 비교 기록:
  - 2026-06-03 기준 제품 확정은 아니며, 오늘의집/시공 사례를 참고해 동화자연마루 진 그란데 `사하라 라이트`, 동화자연마루 진 그란데 `이모션 블랑`, 구정마루 `마뷸러스 젠` 모로칸크림/웜브리즈 계열을 비교했다.
  - 2026-06-04 기준 KCC글라스 홈씨씨 `숲 강마루 스톤 스퀘어`도 비교 후보로 추가했다. 597mm 정사각 스톤 디자인 강마루라 방향성은 맞지만, 출시 초기라 오늘의집/일반 소비자 시공사진 축적이 부족해 샘플·납기·평당 단가 확인 전에는 메인 후보로 올리지 않는다.
  - 사하라 라이트/이모션 블랑은 메인 비교 후보, 마뷸러스 젠은 타일감이 강한 상향 비교 후보, KCC 스톤 스퀘어는 신제품 비교 후보로 본다.
- 확인할 것:
  - 우드 무늬가 과하게 보이지 않는 제품 유무와 실제 샘플 색감
  - 찍힘/스크래치/물 사용 구역 내구성
  - 바닥 수평 상태와 기존 바닥 철거 범위
  - 거실-발코니 경계부 단차 처리
  - 걸레받이/문턱/방문 하부 간섭
  - 자재비/시공비/철거·폐기비/바탕 보정/걸레받이/문턱 마감 포함 여부

## 8. 안방

- 상태: 결정 성향 강함
- 방향: 침실 중심 + 붙박이장 우선
- 구성:
  - 킹 또는 라지킹 침대
  - 긴 벽 쪽 슬라이딩 붙박이장 우선 검토
  - 안방에는 책상을 두지 않는 방향
- 확인할 것:
  - 침대 실제 사이즈
  - 출입문 스윙 방향
  - 창/커튼/콘센트 위치
  - 붙박이장 설치 가능 벽면 길이
  - L자 붙박이장 시 통로 폭

## 9. 작업 공간

- 상태: 결정 성향 강함, 작은방 제작가구 제외 방향으로 업데이트
- 방향: 작은방을 서재/작업방으로 사용
- 현재 방향:
  - 기존 IKEA 1800×900 책상은 정리/매각 검토
  - 1400mm 모션데스크 + 스타일러 + 기존 IKEA 옷장 조합을 기본안으로 검토
  - 모션데스크 폭은 1400mm로 결정
  - 책상은 1400 × 600~700급 우선, 기본 배치 기준은 1400 × 700
  - 1600폭 책상은 기본 후보에서 제외
  - 작은방에 스타일러 자리 확보
  - 기존 IKEA 옷장을 그대로 활용하는 안을 우선 검토
  - 작은방 제작 붙박이장/수납장은 기본안에서 제외하고, 예산 여유가 있을 때만 옵션으로 비교
  - 거실 책상 배치는 우선순위 낮음
- 스타일러 자리 기준:
  - 모델 미확정 상태에서는 우선 W700 × D700 × H2000mm 내외 여유공간을 남기는 방향
  - 최종 제작 전 실제 모델 치수, 문 열림, 콘센트, 환기/방열 여유 확인
- 확인할 것:
  - 모니터 개수
  - PC 본체 위치
  - 1400mm 모션데스크 기준 의자 후퇴 공간과 문열림 간섭
  - 책장/수납장 필요 여부
  - 기존 IKEA 옷장 배치 시 스타일러, 책상, 콘센트, 문열림 간섭 여부
  - 작은방 붙박이장 옵션을 다시 볼 경우 스타일러 공간 확보 가능 여부
  - 콘센트/랜선 위치
  - 의자 뒤 공간

  - 콘센트 위치와 단독 회로 여부

## 11. 안방 앞/화장실 입구 화장대

- 상태: 검토 후보, 선호 있음
- 방향:
  - 안방에서 나와 화장실 입구 쪽 벽면에 작은 화장대 설치 검토
  - 머리 말리는 공간으로도 사용
- 장점:
  - 안방을 더 깔끔하게 유지 가능
  - 화장실/외출 동선과 잘 맞음
  - 20평대에서 독립 파우더존 느낌을 줄 수 있음
- 주의:
  - 습기
  - 통행폭
  - 문 간섭
  - 조명 그림자
  - 콘센트 위치
- 요청 방향:
  - 얕은 깊이의 벽부형/하부장형 화장대
  - 거울 또는 간접조명 거울
  - 드라이기용 콘센트 최소 2구 이상

## 12. 주방 가전/냉장고장

- 상태: 방향 결정 + 실측/설치조건 확인 필요
- 방향:
  - 냉장고장 제작 검토
  - 인덕션 설치
  - 사각 싱크볼 + 거위목 수전 희망
  - 식기세척기는 싱크대 하부장에 설치 희망
  - 아일랜드 식탁/장에는 밥솥, 에어프라이어, 전자레인지 배치 희망
  - 아일랜드 앞쪽에는 로봇청소기 도킹 공간 확보 희망
