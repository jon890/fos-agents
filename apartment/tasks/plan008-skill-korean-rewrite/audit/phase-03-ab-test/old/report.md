## 오늘 결정할 3개

### 1. Q-024 도배 범위와 자재

도배는 어느 범위·자재를 1차 견적 기준으로 잡을까요?

- A. 전체 실크 + 천장 포함 (장기 거주, 마감 품질 우선)
- B. 거실·주방 실크 + 방 합지 + 천장 포함 (예산-품질 절충)
- C. 전체 합지 + 천장 포함 (예산 방어, 갈아타기 전제)

추천: B. "잘 수리해서 살다가 갈아타기" 전제와 화이트/베이지 마감 톤을 동시에 맞추려면, 시야에 많이 들어오는 거실·주방·복도는 실크로 가고 작은방·방은 합지로 묶는 절충안이 예산과 마감을 가장 안정적으로 균형 잡습니다. 천장은 24평 전체에 포함하는 편이 균질감과 균열 리스크에서 유리합니다.

### 2. Q-025 마루 자재 1차 후보

업체에 어떤 마루를 1차 후보로 요청할까요?

- A. 광폭/사각 패턴 강마루 (예: 동화 나투스진 그란데/스퀘어 계열)
- B. 타일 패턴 일반 강마루
- C. SPC/스톤마루

추천: A. D-007의 "600각 타일 느낌을 살리되 강마루의 생활감을 유지" 방향과 가장 잘 맞습니다. 단, 광폭/사각 마루는 재고와 단가 변동이 있으므로 견적서에는 `A 1순위 + B 가격 비교용 1안`을 함께 요청하는 편이 안전합니다. C는 거실 전체보다 발코니/세탁실 같은 물 사용 구역 옵션 비교로만 두는 편이 좋습니다.

### 3. Q-026 작은방 공사 범위

작은방은 작업공간·스타일러·붙박이장 외 어디까지 1차 견적에 넣을까요?

- A. 최소: 도배·바닥재만 (붙박이장·스타일러 자리만 비워둠)
- B. 표준: 도배·바닥재 + 조명·콘센트 위치 재배치 + 붙박이장
- C. 풀세트: 도배·바닥재 + 조명·콘센트 + 붙박이장 + 창호 보수 + 도어 교체

추천: B. 작은방은 1400~1600 모션데스크와 스타일러가 함께 들어가므로 콘센트 위치·조명 그림자가 체감 만족도를 좌우합니다. 도배·바닥재만으로는 작업공간 동선이 나중에 다시 풀려야 할 가능성이 큽니다. 창호·도어 교체는 현장 상태 확인 후 옵션으로 분리해 받는 편이 안전합니다.
