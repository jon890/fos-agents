# Category style map for 지융로그

Use this as the extension layer for `naver-blog-writing`. Keep the base 지융 voice, then apply the closest category module.

## Common module format

Each category should define:

- Reader intent.
- Title formulas.
- Required facts.
- Natural flow.
- Photo slots.
- Search keywords.
- AI-readable entities.
- Tone cautions.

---

## 맛집로그

### Reader intent

Readers want to know whether to visit, what to order, how much it costs, whether there is waiting, and whether parking/location is convenient.

### Title formulas

```text
[지역/역명] 맛집 [상호명] [대표 메뉴] 후기
[지역/역명] [대표 메뉴] 맛집 [상호명] 메뉴, 웨이팅, 주차 후기
[방송/유명 계기] 나온 [지역] 맛집 [상호명] 후기
```

### Required facts

- 상호명, 지역/역명, 대표 메뉴.
- 방문 시간대, 웨이팅 여부.
- 주문 메뉴, 가격.
- 맛, 양, 재방문 의사.
- 주차/예약/위치 if known.

### Flow

인사 → 왜 갔는지 → 외관 → 웨이팅/주차 → 메뉴판 → 주문 메뉴 → 맛 후기 → 볶음밥/디저트 등 마무리 → 추천 대상.

### Photo slots

외관, 입구, 내부, 메뉴판, 기본 반찬, 대표 메뉴, 추가 메뉴, 영수증/지도.

### Keywords

`지역 맛집`, `역명 맛집`, `상호명`, `대표 메뉴`, `메뉴`, `웨이팅`, `주차`, `가격`, `후기`.

### Tone cautions

과하게 미식가처럼 쓰지 않는다. “맛있더라고요”, “양이 많았어요”, “제 입맛에는” 정도가 자연스럽다.

## 카페로그

### Reader intent

Readers want atmosphere, dessert/커피 맛, 좌석, 주차, 사진 찍기 좋은지, 데이트/드라이브에 맞는지.

### Title formulas

```text
[지역] 분위기 좋은 카페 [상호명] [대표 디저트] 후기
[지역] 드라이브 카페 [상호명] 주차, 디저트 후기
[지역] 디저트 맛집 [상호명] 메뉴 추천
```

### Required facts

- 상호명, 지역, 대표 메뉴.
- 좌석/분위기, 주차, 포장 가능 여부.
- 주문한 음료/디저트, 가격.
- 사진 찍기 좋은 포인트.

### Flow

인사 → 카페 방문 이유 → 외관 → 내부/좌석 → 메뉴판 → 주문 → 맛/분위기 → 추천 대상.

### Photo slots

외관, 내부 좌석, 메뉴판, 디저트 쇼케이스, 커피/디저트, 포장, 포토존.

### Keywords

`지역 카페`, `분위기 좋은 카페`, `디저트 맛집`, `브런치 카페`, `주차`, `포장`, `메뉴 추천`.

## 호텔/숙소

### Reader intent

Readers compare location, 가격, 객실 상태, 체크인, 조식, 뷰, 교통, 재방문 의사.

### Title formulas

```text
[지역] 위치 좋은 호텔 [호텔명] [객실 타입] 후기
[여행지] 가성비 숙소 [호텔명] 체크인, 위치, 객실 후기
[지역] [호텔명] 조식, 수영장, 주차 후기
```

### Required facts

- 호텔명, 지역, 객실 타입.
- 숙박 시기, 예약 경로 if known.
- 체크인/체크아웃, 위치/교통.
- 객실 크기, 침대, 욕실, 어메니티.
- 장점과 아쉬운 점.

### Flow

인사 → 여행 맥락 → 예약/위치 → 체크인 → 객실 → 욕실/어메니티 → 부대시설 → 장단점 → 추천 대상.

### Photo slots

외관, 로비, 객실 전체, 침대, 뷰, 욕실, 어메니티, 조식/수영장, 주변 위치.

### Keywords

`지역 호텔`, `위치 좋은 호텔`, `가성비 숙소`, `객실 후기`, `조식`, `주차`, `체크인`, `교통`.

## 여행/이동/관광지

### Reader intent

Readers want route, 비용, 준비물, 웨이팅, 시간 절약, 실제 동선.

### Title formulas

```text
[여행지] [장소명] 가는 방법, 준비물, 웨이팅 후기
[여행지] [명소] 입장권만으로 즐기는 방법, 꿀팁 후기
[공항]에서 [호텔/시내] 가는 방법, 오는 방법
```

### Required facts

- 여행지, 장소명, 방문 날짜/계절 if relevant.
- 이동 방법, 비용, 소요 시간.
- 예약/입장권/준비물.
- 실제 동선과 아쉬운 점.

### Photo slots

출발 위치, 표지판, 티켓/예약 화면, 이동수단, 현장 입구, 내부/체험, 먹거리, 기념품.

### Keywords

`여행지 여행`, `장소명 가는 방법`, `준비물`, `웨이팅`, `입장권`, `꿀팁`, `후기`.

## 제품후기/쇼핑/패션/뷰티

### Reader intent

Readers want 실제 사용감, 사이즈, 색상, 착용샷, 가격, 구매처, 장단점.

### Title formulas

```text
[브랜드] [제품명] [색상/사이즈] 후기, 착용샷
[제품군] 추천 [브랜드 제품명] 실사용 후기
[구매처]에서 산 [브랜드 제품명] 후기
```

### Required facts

- 브랜드, 제품명, 옵션, 가격, 구매처.
- 사이즈/색상/용량.
- 사용 기간 or 착용 상황.
- 장점/아쉬운 점.
- 재구매/추천 대상.

### Photo slots

패키지, 구성품, 상세컷, 사용 전/후, 착용샷, 질감/발색/사이즈 비교.

### Keywords

`제품명 후기`, `브랜드 제품명`, `사이즈`, `착용샷`, `실사용 후기`, `추천`, `가격`, `구매처`.

## 일상/포토로그

### Reader intent

Readers follow 지융의 근황, 맛집/여행/쇼핑 조각, 월간 기록.

### Title formulas

```text
[월] 일상 - [핵심 장소/이벤트 나열]
[계절] 일상 - [여행, 맛집, 쇼핑, 데이트]
```

### Required facts

- 월/기간.
- 주요 장소와 사건.
- 사진별 짧은 코멘트.

### Flow

짧은 인사 → 월간 분위기 → 사진별 조각 기록 → 가벼운 마무리.

### Keywords

`일상`, `일상블로그`, `월간일상`, `직장인일상`, 주요 장소명.

## 주식로그

### Reader intent

20-30대 독자는 어려운 투자 보고서보다 “요즘 왜 오르내리는지”, “초보가 볼 포인트”, “내가 관심 두는 이유”를 원한다.

### Title formulas

```text
[종목명] 주가 왜 움직일까, 초보 투자자가 본 포인트
[업종/테마] 관련주 [종목명] 실적, 이슈 정리
[종목명] 배당/실적/전망 쉽게 정리
```

### Required facts

- 종목명, 티커, 기준 날짜.
- 최근 주가 방향은 확인한 자료 기준으로만.
- 실적, 뉴스, 업종 이슈, 리스크.
- 투자 권유가 아님을 자연스럽게 명시.

### Flow

인사 → 왜 관심 생겼는지 → 최근 이슈 → 쉽게 풀어쓴 사업/실적 → 좋게 보는 점 → 걱정되는 점 → 내가 볼 체크포인트 → 마무리.

### Photo slots

차트 캡처, 실적 표 캡처, 뉴스 헤드라인, 기업 서비스/제품 이미지. 저작권 주의.

### Keywords

`종목명 주가`, `종목명 실적`, `종목명 전망`, `배당`, `관련주`, `리스크`.

### Tone cautions

수익 보장, 매수/매도 권유 금지. 지융 톤으로 쉽고 조심스럽게 쓴다.

## 아파트/부동산

### Reader intent

20-30대 독자는 실거주 관점, 교통, 생활권, 가격 흐름, 인테리어 가능성, 신혼집 관점을 본다.

### Title formulas

```text
[지역] [아파트명] 실거주 관점으로 본 장단점
[지역] 신혼집 후보 [아파트명] 교통, 생활권, 가격 흐름
[아파트명] 인테리어 전 알아본 구조와 분위기
```

### Required facts

- 아파트명, 동네, 연식, 평형/구조 if known.
- 교통, 학군/생활권, 상권, 관리 상태.
- 최근 가격 흐름은 출처 기준으로만.
- 개인적 관심 이유.

### Flow

인사 → 왜 관심 있는지 → 위치/교통 → 생활권 → 단지 분위기 → 평형/구조 → 가격 흐름 → 장점/아쉬운 점 → 이런 사람에게 맞을 듯.

### Photo slots

지도, 단지 입구, 주변 상권, 평면도, 인테리어 참고 이미지. 사생활/차량번호 주의.

### Keywords

`지역 아파트`, `아파트명`, `실거주`, `신혼집`, `교통`, `생활권`, `매매가`, `전세가`, `인테리어`.

### Tone cautions

투자 단정 금지. 개인 사생활 노출 주의. 가격은 날짜와 출처를 함께 쓴다.
