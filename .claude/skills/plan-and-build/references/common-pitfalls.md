# Common Pitfalls — compatibility shim

`common-pitfalls.md` 단일 파일은 `common-pitfalls/` 디렉터리 구조로 분리되었다.

새 작업자는 먼저 [`common-pitfalls/INDEX.md`](common-pitfalls/INDEX.md)를 읽고,
현재 작업의 trigger와 맞는 pattern file만 추가로 읽는다.
어떤 항목이 필요한지 모호하면 해당 category 디렉터리 전체를 확인한다.

기존 task와 문서가 `common-pitfalls.md`를 참조할 수 있으므로 이 shim은 당분간 유지한다.
