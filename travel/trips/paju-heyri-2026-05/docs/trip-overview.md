# 파주 헤이리 반나절 데이트 개요

## 기본 정보
- 여행지: 파주 헤이리 예술마을 / 탄현면 일대
- 일정: 날짜 미정, 반나절 데이트
- 성격: 정식 여행보다는 서울 근교 바람쐬기 + 점심 + 카페 + 그릇 쇼핑

## 확정된 방향
- 점심은 **송우현의 파주 국물없는우동**을 1순위로 둔다.
- 식사 후 **헤이리 예술마을 산책**을 한다.
- 카페는 와이프가 좋아하는 핵심 요소라 여러 후보를 열어두고 더 살펴본다.
- 마지막 쇼핑 후보로 **모드니 프리미엄 본점**을 넣는다.

## 주요 장소
- 송우현의 파주 국물없는우동: <https://map.naver.com/p/search/송우현의%20파주%20국물없는우동>
- 헤이리 예술마을: <https://map.naver.com/p/search/헤이리%20예술마을>
- 모드니 프리미엄 본점: <https://map.naver.com/p/search/모드니%20프리미엄%20본점>

## 링크 우선순위
1. 네이버 지도: 위치, 영업시간, 휴무, 전화, 리뷰 확인
2. 네이버 블로그: 실제 방문 후기, 사진, 동선 확인
3. 공식 홈페이지/관광공사/기타 블로그: 보조 자료

## 재확인 필요
- 방문 날짜 확정
- 송우현의 파주 국물없는우동 휴무/브레이크타임/웨이팅
- 카페 최종 선택
- 모드니 방문 여부와 마감 시간
