# 파주 헤이리 데이트 의사결정 로그

## 2026-05-22

### 결정 주제
- 서울 근교에서 가볍게 바람쐴 만한 데이트 코스 선정
- 파주 헤이리 예술마을 중심의 반나절 코스 구성
- 일정표와 후보 정보를 오사카 여행처럼 travel 워크스페이스에 기록

### 반영한 결정
1. 목적지
   - 여러 후보 중 **파주 헤이리 예술마을**로 결정했다.
   - 정식 여행보다는 가벼운 근교 데이트로 관리한다.

2. 코스 방향
   - 초기 1번 코스를 기반으로 간다.
   - 현재 코스는 **송우현의 파주 국물없는우동 -> 헤이리 예술마을 산책 -> 분위기 좋은 카페 -> 모드니 프리미엄 본점** 흐름이다.

3. 점심
   - 잇탈리는 동선용 무난한 이탈리안으로 판단했고, 맛집 목적이면 기대치를 낮추는 편이 낫다고 정리했다.
   - 지융이 제안한 **송우현의 파주 국물없는우동**이 맛집 목적에 더 강해 보여 점심 1순위로 올렸다.

4. 카페
   - 와이프가 카페를 좋아하므로 카페는 한 곳으로 바로 확정하지 않고 후보를 넓게 남긴다.
   - 현재 후보 전체: 화이트블럭, 카메라타, 커피공장103, 카페 귀천, 동화속으로, 프로방스 베이커리 앤 카페, 말똥도넛, 더티트렁크.
   - 우선순위 초안은 화이트블럭 -> 카메라타 -> 커피공장103 -> 카페 귀천이다.

5. 쇼핑
   - 그릇/주방용품 구경 후보로 **모드니 프리미엄 본점**을 포함한다.
   - 모드니는 19시 마감으로 파악했으므로 너무 늦게 넣지 않는다.

6. 참고 링크 원칙
   - 네이버 지도와 네이버 블로그를 우선한다.
   - 기타 링크는 네이버 쪽 정보가 부족할 때 보조로만 사용한다.

### 남은 결정
- 방문 날짜와 출발 시간
- 점심을 송우현으로 확정할지, 웨이팅이 부담되면 대체 식당을 둘지
- 메인 카페 1곳과 백업 카페 2곳
- 모드니를 필수로 볼지, 컨디션 따라 선택으로 둘지

### 다음 확인 후보
1. 송우현의 파주 국물없는우동 당일 영업/휴무/브레이크타임
2. 카페별 실제 분위기와 최근 후기
3. 헤이리 예술마을 내 주차와 이동 동선
4. 모드니 영업시간과 주차
